### **Karvinen 2026: SSH Public Key – Login Without Password**
(Karvinen 2026a)

**Basics of SSH**
- Install SSH:
  ```bash
  sudo apt-get update
  sudo apt-get -y install ssh
  ```
- Enable + start service:
  ```bash
  sudo systemctl enable --now ssh
  #-- Now starts it now
  ```
- Generate SSH keypair:
  ```bash
  ssh-keygen
  ```
  Don't shoot yourself in the foot.Don't overwrite keypair. Not fun.

### **Karvinen 2026: Hello Ansible**
(Karvinen 2026b)

- **Ansible = Infrastructure as Code (IaC)**  
  Define the desired state; Ansible only changes what is needed.
- **Agentless**  
  Uses SSH. Managed by SSH daemon + Python.
- **Controller vs. Managed Nodes**  
  Controller runs Ansible; connects to nodes via SSH.
- **Playbooks define roles**  
  Assigns roles to machines.

### First Fail

```

### Working Result

```
cat roles/firewall/tasks/main.yml
```
```
- name: Install utf
  apt:
    name: ufw
    state: present
  become: true

- name: Reset
  ufw:
    state: reset
  become: true

- name: Deny incoming
  ufw:
    default: deny
    direction: incoming
  become: true

- name: Allow outgoing
  ufw:
    default: allow 
    direction: outgoing
  become: true

- name: Allow ssh
  ufw:
    rule: allow
    name: OpenSSH
  become: true

- name: Allow HTTP and HTTPS
  ufw:
    rule: allow
    port: "{{ item }}"
    protocol: tcp
  loop:
    - 80
    - 443
  become: true

- name: Start ufw and logging
  ufw:
    state: enabled
    logging: on
  become: true
```
