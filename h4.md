## 4.12.1 

Domain Specific Languages (DSLs) used in configuration management can become very large and complex.

For example, Salt provides over 500 state functions and extensive documentation exceeding 20 000 lines, over 75 000 words. 

Puppet is smaller in number of built-in functions (around 100+), 

“simple” configuration management tools end up being large systems with many functions and custom logic that users must learn.

---

## 4.12.2 
DSLs real-world usage tends to rely on a small subset.

In practice, most configurations use common patterns such as:
- package installation
- file management
- service management
- basic control flow (if/case)
- user and group management

Studies of production Puppet configurations show that a small number of functions cover the majority of use cases. 

---


## 4.12.3.1 

A key property of these systems is **idempotency**, meaning that running the same configuration repeatedly should not change the system if it is already in the correct state.

In practice, most configuration tools rely heavily on:
- file manipulation
- system command execution

## Reddit saves the day

Eventually I found a Reddit guide that actually worked (few months old) :

https://www.reddit.com/r/Ubiquiti/comments/1qgscs1/how_to_guide_installing_unifi_network_application/

The Difference: correct MongoDB repo + correct UniFi repo setup.

### I ran the guide step by step (copy from from Reddit)

```
# Update system
sudo apt update && sudo apt upgrade -y && sudo apt autoremove -y

# Required tools
sudo apt install -y gnupg curl ca-certificates

# Keyring directory
sudo install -d -m 0755 /etc/apt/keyrings

# MongoDB key
curl -fsSL https://www.mongodb.org/static/pgp/server-8.0.asc | sudo gpg --dearmor -o /etc/apt/keyrings/mongodb-server-8.0.gpg
sudo chmod 0644 /etc/apt/keyrings/mongodb-server-8.0.gpg

# MongoDB repo
sudo tee /etc/apt/sources.list.d/mongodb-org-8.0.sources > /dev/null <<'EOF'
Types: deb
URIs: https://repo.mongodb.org/apt/debian
Suites: bookworm/mongodb-org/8.0
Components: main
Architectures: amd64
Signed-By: /etc/apt/keyrings/mongodb-server-8.0.gpg
Enabled: true
EOF

# UniFi key
curl -fsSL https://dl.ui.com/unifi/unifi-repo.gpg | sudo tee /etc/apt/keyrings/unifi-repo.gpg > /dev/null
sudo chmod 0644 /etc/apt/keyrings/unifi-repo.gpg

# UniFi repo
sudo tee /etc/apt/sources.list.d/unifi.sources > /dev/null <<'EOF'
Types: deb
URIs: https://www.ui.com/downloads/unifi/debian
Suites: stable
Components: ubiquiti
Architectures: amd64
Signed-By: /etc/apt/keyrings/unifi-repo.gpg
Enabled: true
EOF

# Install
sudo apt update
sudo apt install -y mongodb-org
sudo apt install -y unifi
```

## Ansible Role Breakdown
- I ran these one block at a time. There were a few typos and formatting issues along the way, but I’ve left those out of the report since they don’t really add anything useful.
- ✔️ = I ran the command ansible-playbook unifi.yml successfully.

### 1. Install required packages

```
- name: Install required packages
  apt:
    name:
      - gnupg
      - curl
      - ca-certificates
    state: present
```
✔️
---

### 2. Create keyring directory

```
- name: Create keyring directory
  file:
    path: /etc/apt/keyrings
    state: directory
    mode: "0755"
```
✔️
---

### 3. Download MongoDB key

```
- name: Download MongoDB GPG key
  get_url:
    url: https://www.mongodb.org/static/pgp/server-8.0.asc
    dest: /tmp/mongodb-server-8.0.asc
    mode: "0644"
```
✔️
---

### 4. Convert key + add repo

```
- name: Convert MongoDB key to keyring format
  command: gpg --dearmor -o /etc/apt/keyrings/mongodb-server-8.0.gpg /tmp/mongodb-server-8.0.asc
  args:
    creates: /etc/apt/keyrings/mongodb-server-8.0.gpg

- name: Add MongoDB repository
  copy:
    dest: /etc/apt/sources.list.d/mongodb-org-8.0.sources
    mode: "0644"
    content: |
      Types: deb
      URIs: https://repo.mongodb.org/apt/debian
      Suites: bookworm/mongodb-org/8.0
      Components: main
      Architectures: amd64
      Signed-By: /etc/apt/keyrings/mongodb-server-8.0.gpg
      Enabled: true
```
✔️
---

### 5. UniFi repo setup

```
- name: Download UniFi GPG key
  get_url:
    url: https://dl.ui.com/unifi/unifi-repo.gpg
    dest: /etc/apt/keyrings/unifi-repo.gpg
    mode: "0644"

- name: Add UniFi repository
  copy:
    dest: /etc/apt/sources.list.d/unifi.sources
    mode: "0644"
    content: |
      Types: deb
      URIs: https://www.ui.com/downloads/unifi/debian
      Suites: stable
      Components: ubiquiti
      Architectures: amd64
      Signed-By: /etc/apt/keyrings/unifi-repo.gpg
      Enabled: true
```
✔️
---

### 6. Install packages

```
- name: Update apt cache
  apt:
    update_cache: yes

- name: Install MongoDB and UniFi
  apt:
    name:
      - mongodb-org
      - unifi
    state: present
```
✔️
