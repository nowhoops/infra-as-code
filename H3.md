### Fix
Made new hometest.site.conf
``` bash
cd /etc/apache2/sites-available
sudoedit hometest.site.conf
``` 
- cat hometest.site.conf
``` yml
<VirtualHost *:80>
    ServerName hometest.site
    DocumentRoot /home/azian/sites

    <Directory /home/azian/sites/>
        Require all granted
    </Directory>
</VirtualHost>
```
--- 

``` bash
sudo systemctl reload apache2
```
I needed to set permissions:

``` bash
cd ~/sites
sudo chmod ugo+x .
sudo systemctl reload apache2
```
