# c) Automatisointi (Ansible + Nginx)

## Goal
Automate Nginx installation using Ansible.  
Web pages (HTML) are still created manually.

---

## 1. Playbook

nano nginx.yml

``` yaml
- hosts: all
  become: true
  roles:
    - nginx
```

---

## 2. Role setup

cp -R app nginx

nano main.yml

``` yaml
start by installing nginx

- apt:
    name:
      - nginx
    state: present

- service:
    name: nginx
    state: started
    enabled: yes
```

Run playbook:

``` bash
ansible-playbook nginx.yml
```

---

## 3. Variables

mkdir nginx/vars/
nano main.yml

``` yaml
web_user: "azian"
web_root: "/home/{{ web_user }}/nginx"
```

---

## 4. Create web directory

nano nginx/tasks/main.yml

``` yaml
- name: Create dir
  file:
    path: "{{ web_root }}"
    state: directory
    owner: "{{ web_user }}"
    group: "{{ web_user }}"
    mode: '0755'
```

Success.

---

## 5. Create test index.html

``` yaml
- name: Create index.html
  copy:
    dest: "{{ web_root }}/index.html"
    content: "<h1>whoop</h1>"
    owner: "{{ web_user }}"
    group: "{{ web_user }}"
    mode: '0644'
```

Success.

---

## 6. Configure Nginx site

``` yaml
- name: Configure nginx site
  copy:
    dest: /etc/nginx/sites-available/default
    content: |
      server {
          listen 80 default_server;
          listen [::]:80 default_server;

          root {{ web_root }};
          index index.html;

          location / {
              try_files $uri $uri/ =404;
          }
      }
```

---

## 7. Restart Nginx

``` yaml
- name: Restart nginx
  service:
    name: nginx
    state: restarted
    enabled: yes
```

Everything went quite smoothly.

I had a few typos I had to fix, but otherwise it worked.
